# Quantum Nexus v7 — Research Cockpit

**Quantum Nexus v7** is an advanced browser-based quantum-neural simulation interface built as a single HTML application. It combines a cyberpunk neural UI, quantum-inspired state simulation, live visual analytics, research-inspired controls, chat assistance, voice commands, experiment recording, and export tools.

This project is designed as an experimental playground for exploring ideas from quantum machine learning, neural interfaces, noisy intermediate-scale quantum systems, hybrid AI control, circuit optimization, and interactive scientific visualization.

> This is an educational and creative simulation. It does not connect to real quantum hardware by default.

### Quantum Gate System

Includes the original gates and expanded v7 controls:

- **H** — Hadamard-style superposition boost
- **X** — Bit-flip activation toggle
- **Y** — Phase/energy perturbation
- **Z** — Coherence phase shift
- **S** — Phase gate
- **T** — Fine phase gate
- **RX** — Rotation around X axis
- **RY** — Rotation around Y axis
- **RZ** — Rotation around Z axis
- **CNOT** — Two-node entanglement operation
- **SWAP** — Node-state swap operation
- **Measure** — Collapse-style measurement

### Built-In Experiment Modes

Quantum Nexus includes preset experiments inspired by common quantum computing concepts:

- **Bell Pair Mode**  
  Creates a two-qubit entanglement-style setup.

- **GHZ Mode**  
  Creates a multi-qubit correlated state.

- **QAOA-Style Mode**  
  Demonstrates hybrid optimization-inspired circuit behavior.

- **Quantum Dot Pulse Mode**  
  Adds optoelectronic/neural-interface-inspired pulse behavior.

- **Stress Test Mode**  
  Rapidly applies operations to test visualization and metric behavior.

- **Auto Pilot Mode**  
  Lets the system automatically explore gates, nodes, and simulated optimization behavior.
